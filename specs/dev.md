# 개발 정책

## 코드 주석 언어

설명 주석은 한국어 중심. 식별자·타입명·키 이름 등 코드와 직접 연결되는
용어는 그대로 두고, 설명 문장은 한국어로 쓴다.

### 규칙
- 함수 설명 주석은 한국어.
- 구현 의도 `//` 주석도 한국어.
- `HUD`, `XP`, `QuestState`, `PlayerActedEvent` 같은 식별자·타입명은 유지.
- 알고리즘 약어·고유명사는 원문 + 한국어 설명.
- ASCII 도식·테스트 도면은 의미가 깨지지 않는 선에서 한국어 설명 보강.

### 검증
- 주석 변경 후 전체 회귀: `cargo test`
- 공백 검사: `git diff --check`

### 남은 개선 후보
- `assets/quests/*.ron` 상단의 타입 나열 번역 기준
- 공개 API 문서 주석 형식 (`목적/인수/반환값`) 통일
- CI 에서 주석 언어 자동 검사
