# 장비 시스템

## 목적

무기·방어구 장착으로 전투 스탯에 영향을 주고, 인벤토리에서 아이템을 관리한다.

## 동작 명세

### 아이템 획득
- [ ] 무기·방어구 획득 시 인벤토리에 개별 아이템으로 추가된다 (검 2개 = 검, 검)
- [ ] 소모품(체력 물약) 획득 시 동일 종류끼리 수량이 누적된다 (x2, x3...)
- [ ] 획득 시 로그 메시지가 출력된다

### 장착 슬롯 (미니맵 아래 항상 표시)
- [ ] 미니맵 컨테이너 아래에 현재 장착 무기·방어구가 항상 표시된다
- [ ] 슬롯은 RPG Awesome 아이콘(검/창/활/방패) + 아이템 이름 2섹션으로 렌더링된다
- [ ] 미장착 슬롯은 아이콘이 흐리게(alpha 0.3) 표시되고 이름은 `-`로 표시된다
- [ ] 장착 시 아이콘이 밝아지고(alpha 0.9) 아이템 이름이 표시된다
- [ ] M키로 미니맵과 함께 숨겨진다

### 장비 관리 패널 (E 키 토글)
- [ ] `E` 키로 장비 패널을 열고 닫는다
- [ ] 패널이 열린 동안 플레이어 이동은 차단된다
- [ ] `↑` `↓` 로 인벤토리 항목을 선택한다
- [ ] `Enter` 로 선택한 무기·방어구를 장착/해제한다
- [ ] `Enter` 로 선택한 소모품을 사용한다
- [ ] `Esc` 또는 `E` 로 패널을 닫는다
- [ ] 현재 장착된 아이템에는 `[장착]` 표시가 된다
- [ ] 패널 내 현재 장착 요약 줄과 인벤토리 각 행에 RPG Awesome 아이콘이 표시된다
- [ ] 아이콘과 한글 이름은 각각 다른 폰트 섹션으로 혼합 렌더링된다

### 전투 스탯
- [ ] 무기 장착 시 공격력이 해당 무기의 값으로 대체된다
- [ ] 방어구 장착 시 기본 방어력에 보너스가 추가된다
- [ ] 장비 해제 시 기본 스탯으로 복원된다

## 아이템 수치

| 종류       | 글리프 | 공격력 | 방어보너스 | 색상     |
|------------|--------|--------|------------|----------|
| 검         | `/`    | 7      | -          | 노란색   |
| 창         | `\|`   | 9      | -          | 노란색   |
| 활         | `)`    | 5      | -          | 노란색   |
| 가죽 갑옷  | `]`    | -      | +2         | 파란색   |
| 체력 물약  | `!`    | -      | -          | 초록색   |

## 드롭 테이블

| 몬스터 | 체력 물약 | 검    | 창    | 활    | 가죽 갑옷 |
|--------|-----------|-------|-------|-------|-----------|
| 고블린 | 30%       | 15%   | -     | -     | -         |
| 오크   | 40%       | -     | 20%   | -     | 10%       |
| 트롤   | 50%       | -     | -     | 25%   | 20%       |

## 기본 스탯 (비무장)

- 무기 미장착: `PLAYER_ATK = 5`
- 방어구 미장착: `PLAYER_DEF = 1`
