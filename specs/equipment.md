# 장비 시스템

## 목적

무기·방어구 장착으로 전투 스탯에 영향을 주고, 인벤토리에서 아이템을 관리한다.

## 동작 명세

### 아이템 획득
- [x] 무기·방어구 획득 시 인벤토리에 개별 아이템으로 추가된다 (검 2개 = 검, 검)
- [x] 소모품(체력 물약) 획득 시 동일 종류끼리 수량이 누적된다 (x2, x3...)
- [x] 획득 시 로그 메시지가 출력된다

### 장착 슬롯
장착 정보는 E 키 장비 패널(/ P A R T S / 섹션)에서 확인한다.
미니맵 아래 별도 슬롯 표시는 제거됐다.

### 장비 관리 패널 (E 키 토글, Cogmind 스타일 우측 패널)

패널은 화면 우측에 고정 너비(260px)로 배치되며 E 키로 토글된다.
Cogmind 게임의 인터페이스를 참고한 녹색 계열 색상 테마를 사용한다.

#### 레이아웃
```
/ P A R T S /
─────────────────────
 W e a p o n
  [아이콘] 검  (|||||||)  ATK 7
 A r m o r
  [아이콘] 없음

/ I N V E N T O R Y /
─────────────────────
 >1 [아이콘] 검  (ATK 7)   [장착]
  2 [아이콘] 창  (ATK 9)
  3 [아이콘] 체력 물약  x2
─────────────────────
↑↓ 이동  Enter 장착/사용  Esc·E 닫기
```

#### 동작 명세
- [x] `E` 키로 우측 패널을 열고 닫는다
- [x] 패널이 열린 동안 플레이어 이동은 차단된다
- [x] `↑` `↓` 로 인벤토리 항목을 선택한다 (커서는 노란색으로 강조)
- [x] `Enter` 로 선택한 무기·방어구를 장착/해제한다
- [x] `Enter` 로 선택한 소모품을 사용한다
- [x] `Esc` 또는 `E` 로 패널을 닫는다
- [x] PARTS 섹션: 현재 장착된 무기·방어구를 아이콘+이름+상태바 형식으로 표시
- [x] PARTS 섹션: 미장착 슬롯은 아이콘과 "없음" 텍스트를 어둡게 표시
- [x] PARTS 섹션: 장착 아이템의 상태바는 ATK/DEF 수치만큼 `|` 문자로 채운다
- [x] INVENTORY 섹션: 번호(1, 2, 3…) + 아이콘 + 이름 + 수치로 나열
- [x] INVENTORY 섹션: 현재 장착된 아이템에 `[장착]` 태그 표시 (밝은 녹색)
- [x] 아이콘(rpg-awesome)과 한글 이름(NanumSquareNeo)은 별도 TextSection으로 혼합 렌더링

### 전투 스탯
- [x] 무기 장착 시 공격력이 해당 무기의 값으로 대체된다
- [x] 방어구 장착 시 기본 방어력에 보너스가 추가된다
- [x] 장비 해제 시 기본 스탯으로 복원된다

## 아이템 수치

| 종류       | 글리프 | 공격력 | 방어보너스 | 색상     |
|------------|--------|--------|------------|----------|
| 검         | `/`    | 7      | -          | 노란색   |
| 창         | `\|`   | 9      | -          | 노란색   |
| 활         | `)`    | 5      | -          | 노란색   |
| 가죽 갑옷  | `]`    | -      | +2         | 파란색   |
| 체력 물약  | `!`    | -      | -          | 초록색   |

## 드롭 테이블

| 몬스터 | 체력 물약 | 검    | 창    | 활    | 가죽 갑옷 |
|--------|-----------|-------|-------|-------|-----------|
| 고블린 | 30%       | 15%   | -     | -     | -         |
| 오크   | 40%       | -     | 20%   | -     | 10%       |
| 트롤   | 50%       | -     | -     | 25%   | 20%       |

## 기본 스탯 (비무장)

- 무기 미장착: `PLAYER_ATK = 5`
- 방어구 미장착: `PLAYER_DEF = 1`
