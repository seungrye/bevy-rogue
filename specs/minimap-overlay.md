# 미니맵 오버레이

## 목적

게임 화면 우측 상단에 반투명 다이아몬드 미니맵을 오버레이로 표시하여 플레이어가 현재 위치와 주변 지형을 즉시 파악할 수 있게 한다.

## 동작 명세

- [x] 미니맵은 게임 화면 우측 상단에 반투명 오버레이로 표시된다 (우측 5px 여백)
- [x] 다이아몬드 형태로 테두리가 표시된다
- [x] 탐험한 타일은 밝기로 구분되어 표시된다 (시야 내 > 탐험됨 > 미탐험)
- [x] 플레이어 위치는 노란색 점으로 표시된다
- [x] `M` 키로 미니맵 표시/숨김 전환이 가능하다 (생성기 이름·힌트 포함 전체 숨김)
- [x] 기본값은 표시 상태이다 (게임 시작 시 미니맵 표시됨)
- [x] 미니맵 아래에 현재 생성기 이름이 청록색으로 표시된다
- [x] 생성기 이름 아래에 `[Tab] 맵 전환` 힌트가 회색으로 표시된다
- [x] `Ctrl`+`=`(`+`) 또는 `Ctrl`+Numpad`+`로 미니맵을 확대한다
- [x] `Ctrl`+`-` 또는 `Ctrl`+Numpad`-`로 미니맵을 축소한다
- [x] 확대/축소 단위는 20px이며 최솟값 80px, 최댓값 280px으로 제한된다

## 엣지 케이스

- 미니맵 숨김 상태에서 다시 M 키를 누르면 다시 표시된다
- 미니맵이 숨겨진 상태에서도 내부 텍스처는 계속 업데이트된다 (재표시 시 최신 상태)
- 맵 경계 밖 영역은 어두운 배경으로 표시된다

## 위치·크기 상수

- 우측 여백: `5.0` px
- 상단 여백: `10.0` px
- 기본 표시 크기: `MINIMAP_DISPLAY_SIZE = 180.0` px (`MinimapConfig::display_size` 초깃값)
- 최솟값: `MINIMAP_MIN_SIZE = 80.0` px
- 최댓값: `MINIMAP_MAX_SIZE = 280.0` px
- 줌 단위: `MINIMAP_ZOOM_STEP = 20.0` px
