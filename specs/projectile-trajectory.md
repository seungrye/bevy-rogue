# 투사체 궤적 시스템

## 목적

활(Bow)을 장착한 상태에서 마우스로 FOV 내 몬스터를 클릭하면,
`bevy_rapier2d` 물리 엔진으로 중력이 적용된 포물선 궤적의 화살을 발사한다.
시각적 궤적은 물리 엔진이 담당하고, 피해 판정은 타일 좌표 비교로 처리한다.

## 동작 명세

- [ ] 활을 장착한 상태에서 FOV 내 몬스터 타일을 클릭하면 화살을 발사한다
- [ ] 화살은 발사자 위치에서 목표 타일을 향해 포물선 궤적으로 날아간다
- [ ] 화살은 중력(`GravityScale(1.5)`)의 영향을 받아 자연스러운 낙하 곡선을 그린다
- [ ] 화살은 진행 방향으로 회전하여 항상 속도 벡터를 향한다
- [ ] 화살이 몬스터 타일에 진입하면 피해를 입히고 즉시 소멸한다
- [ ] 화살이 벽 타일에 진입하면 소멸한다 (관통 불가)
- [ ] 화살은 최대 2초가 지나면 자동 소멸한다 (수명 만료)
- [ ] 화살 발사는 턴을 소비한다 (`PlayerActedEvent` 발행)
- [ ] 활의 유효 사거리는 FOV 반경 8타일과 동일하다
- [ ] 원소 시스템과 연동된다 — 활은 번개 원소를 40% 확률로 부여한다 (기존 동작)

## 상수

- `BOW_RANGE = 8` (타일, FOV 반경과 동일)
- `ARROW_LIFETIME = 2.0` (초, 수명 상한)
- `ARROW_SIZE = Vec2::new(8.0, 3.0)` (픽셀, 가로×세로)
- `ARROW_COLOR = Color::rgb(0.8, 0.6, 0.2)` (황갈색)
- `GRAVITY_SCALE = 1.5` (Rapier 중력 배율)
- 화살 Z 레이어: `0.9` (몬스터(0.8) 위)

## 발사 속도 계산

```
g = gravity_scale × 9.81 × pixels_per_meter
flight_time = 0.3 + distance / 400.0

vx = Δx / flight_time
vy = (Δy + 0.5 × g × flight_time²) / flight_time
```

## 트리거 조건

마우스 왼쪽 클릭 시:
1. 활 장착 상태
2. 클릭 타일에 몬스터 존재 (`MonsterTiles` 참조)
3. 클릭 타일이 FOV 내에서 가시(`map.tiles[idx].visible == true`)
4. 클릭 타일이 사거리 8타일 이내

위 조건 미충족 시 기존 경로 탐색 동작.

## 엣지 케이스

- 인접 타일 몬스터에게도 화살 발사 가능 (근거리 사격)
- 화살이 날아가는 도중 몬스터가 해당 타일을 떠나면 빗나감
- 활을 장착하지 않은 상태에서는 기존 근접 공격 동작 유지
- 모달 UI(장비창·상점·도움말) 열린 상태에서는 발사 불가
