# 미니맵 마커 단일 픽셀화

## 목적

미니맵 마커가 십자 형태로 5 픽셀을 차지해 너무 크게 보이는 문제를 해결한다.
마커 색상이 이미 충분히 구분되므로 1 픽셀만으로도 가독성이 충분하다.

## 현재 동작

`marker_stamp_pixels` 가 중심 + 상하좌우 4 픽셀, 총 5 픽셀을 반환:

```rust
[(x, y), (x - 1, y), (x + 1, y), (x, y - 1), (x, y + 1)]
```

미니맵 픽셀 밀도가 높지 않아 (~1 px/tile) 이 십자가 미니맵에서 큰 영역을
차지한다.

## 동작 명세

- [ ] `marker_stamp_pixels` 제거 또는 단일 픽셀만 반환하도록 변경
- [ ] 호출부에서 직접 `write_minimap_pixel` 호출
- [ ] 기존 십자 가정 테스트 갱신

## 테스트 전략

- 마커 1 픽셀이 정확히 marker_pixel_coords 위치에 그려지는지 검증
