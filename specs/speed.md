# 이동 속도 (Speed)

## 목적

모든 엔티티(플레이어, 몬스터, NPC)가 개별 이동 속도를 갖고, 한 턴에 행동 횟수를 결정한다.
고블린처럼 빠른 적은 플레이어보다 더 자주 움직이고, 트롤 같은 느린 적은 더 적게 움직인다.

## 동작 명세

### Speed 컴포넌트

- [x] `Speed { value: f32, energy: f32 }` — `value`는 기준 속도, `energy`는 누적 에너지
- [x] 플레이어 행동(`PlayerActedEvent`) 시 모든 NPC/몬스터의 `energy += value`
- [x] `energy >= 1.0`인 동안 행동 1회 소비 후 `energy -= 1.0`
- [x] 속도 1.0: 매 턴 1회 행동
- [x] 속도 2.0: 매 턴 최대 2회 행동
- [x] 속도 0.5: 2턴마다 1회 행동

### 엔티티별 기본 속도

| 엔티티 | 속도 | 설명 |
|--------|------|------|
| 플레이어 | 1.0 | 기준값 |
| 고블린 | 1.5 | 빠르고 민첩 |
| 오크 | 1.0 | 일반 |
| 트롤 | 0.5 | 느리고 강함 |
| 장로 | 0.5 | 노쇠함 |
| 연금술사 | 0.75 | 조심스러운 걸음 |
| 촌장 | 1.0 | 보통 |
| 상인 | 1.0 | 보통 |
| 농부 | 0.75 | 느릿느릿 |
| 아이 | 1.5 | 뛰어다님 |
| 노인 | 0.5 | 천천히 걸음 |

## 엣지 케이스

- `just_bumped` 상태의 주민은 에너지는 누적되지만 그 턴에는 행동하지 않는다
- 속도가 비정수일 때 에너지가 1.0을 넘으면 남은 에너지는 다음 턴으로 이월된다
- 공격도 행동 1회를 소비한다 (속도 2.0 몬스터는 같은 턴에 두 번 공격 가능)
