# 아이템 드롭

## 목적

몬스터 처치 시 확률적으로 소모품·장비 아이템을 드롭하여 전투 보상을 제공한다.
아이템 종류는 `ItemKind` enum으로 관리하고, 장비 수치와 인벤토리 동작은 `specs/equipment.md`와 공유한다.

## 동작 명세

- [x] 몬스터 처치 시 `ItemDropEvent`가 발생하고, 몬스터별 드롭 테이블을 독립 확률로 롤한다
- [x] 드롭된 아이템은 처치된 몬스터의 타일 위치에 현재 글리프 스타일로 표시된다
- [x] 아이템은 Z 레이어 `0.3`에 위치한다 (바닥(0) 위, 핏자국(0.5)·몬스터(0.8) 아래)
- [x] 플레이어가 아이템 타일로 이동하면 자동 수집된다
- [x] 수집 시 인벤토리에 추가되고 로그 메시지가 출력된다
- [x] 체력 물약은 소모품 스택에 누적되며, 장비 패널에서 사용 시 HP `POTION_HEAL = 8` 회복 (최대 HP 초과 불가)
- [x] 무기·방어구는 개별 인벤토리 아이템으로 추가된다

## 드롭 테이블

각 항목은 서로 독립적으로 판정된다. 예를 들어 오크는 체력 물약과 창을 같은 처치에서 동시에 떨어뜨릴 수 있다.

| 몬스터 | 체력 물약 | 검 | 창 | 활 | 가죽 갑옷 |
|--------|-----------|----|----|----|-----------|
| 고블린 | 30% | 15% | - | - | - |
| 오크 | 40% | - | 20% | - | 10% |
| 트롤 | 50% | - | - | 25% | 20% |
| 기타 | 25% | - | - | - | - |

## 아이템 목록

| 이름 | 종류 | 글리프 | 색상 | 효과 |
|------|------|--------|------|------|
| 체력 물약 | 소모품 | `!` | 초록색 | HP +8 |
| 검 | 무기 | `/` | 노란색 | ATK 7 |
| 창 | 무기 | `|` | 노란색 | ATK 9 |
| 활 | 무기 | `)` | 노란색 | ATK 5 |
| 가죽 갑옷 | 방어구 | `]` | 파란색 | DEF +2 |

## 상수

- `POTION_HEAL = 8` (HP)
- `Z_ITEM = 0.3`

## 엣지 케이스

- 같은 타일에 여러 아이템이 있으면 한 번에 하나씩 수집된다
- 이미 HP가 최대인 경우 체력 물약을 써도 넘치지 않는다
- 퀘스트 아이템은 몬스터 드롭이 아니라 퀘스트 스폰 시스템에서 관리한다
