# 전투

근접·원격 전투, 데미지·피드백, 원소 반응, 속도, 투사체, 파티클.

## 적·근접 전투

"인접 타일로 이동 시도 = 공격" 규칙으로 이동·전투 통합.

### 스폰
- 던전 맵에만 (마을·숲엔 X).
- 방 (room) 중심 타일에 스폰. 플레이어 스폰 방 (`rooms[0]`) 제외.
- 맵 재생성 시 기존 몬스터 모두 제거 후 재스폰.

### 전투 스탯
- 플레이어 초기: `PLAYER_HP=30`, `PLAYER_ATK=5`, `PLAYER_DEF=1`,
  `PLAYER_MP=10`.
- 데미지 = `max(1, 공격자.공격력 - 방어자.방어력)`.
- 몬스터 유형:

| 글리프 | 색상 | 이름 | HP | 공격 | 방어 |
|--------|------|------|----|------|------|
| `g` | 연두 | 고블린 | 6 | 3 | 0 |
| `O` | 주황 | 오크 | 10 | 5 | 2 |
| `T` | 청록 | 트롤 | 16 | 8 | 3 |

### 공격
- 플레이어가 몬스터 타일로 이동 시도 → 근접 공격 (이동 안 함).
- 몬스터 인접 시 (4방향) 플레이어 공격 (이동 안 함).
- 공격 결과 (데미지, 남은 HP) 로그.
- 플레이어 공격도 턴 소비.

### 몬스터 AI (턴제)
- 플레이어 턴 종료 후 살아있는 몬스터가 각각 한 번 행동.
- 인접 시 공격, 아니면 플레이어 방향으로 한 칸 이동 (Manhattan 최소화).
- 다른 몬스터 점유 타일 회피, 이동 가능 타일 없으면 제자리.

### 사망
- HP ≤ 0 → 다음 정리 단계에서 제거.
- 몬스터 사망: "[이름]을(를) 처치했다!"
- 플레이어 사망: "당신은 죽었습니다." + 이동 불가 (`Defeated`).
- 같은 턴에 죽은 몬스터는 그 턴 행동 건너뜀.

### 구현 범위
- `CombatStats { hp, max_hp, mp, max_mp, attack, defense }` —
  플레이어/몬스터 공통 (`combat`).
- `Defeated` — 사망 마커.
- `calc_damage(attack, defense) -> i32` — 순수 함수.
- `MonsterTiles` 리소스 — PreUpdate 동기화, 이동 차단.
- `AttackMonsterEvent(x, y)` — 충돌 시 발행.
- `MonsterRespawnEvent { map_type, rooms }` — 맵 재생성 시.

## 속도 (Speed)

엔티티별 이동 속도로 한 턴 행동 횟수 결정.

### `Speed { value: f32, energy: f32 }`
- 플레이어 행동 (`PlayerActedEvent`) 시 모든 NPC/몬스터 `energy += value`.
- `energy >= 1.0` 동안 행동 1회 + `energy -= 1.0`.
- 1.0 → 매 턴 1회, 2.0 → 매 턴 최대 2회, 0.5 → 2턴마다 1회.

### 기본 속도

| 엔티티 | 속도 | 설명 |
|--------|------|------|
| 플레이어 | 1.0 | 기준 |
| 고블린 | 1.5 | 빠름 |
| 오크 | 1.0 | 보통 |
| 트롤 | 0.5 | 느리고 강함 |
| 장로 | 0.5 | 노쇠 |
| 연금술사 | 0.75 | 조심 |
| 촌장 | 1.0 | 보통 |
| 상인 | 1.0 | 보통 |
| 농부 | 0.75 | 느릿 |
| 아이 | 1.5 | 뛰어다님 |
| 노인 | 0.5 | 천천히 |

### 엣지
- `just_bumped` 주민은 에너지 누적되지만 그 턴 행동 안 함.
- 비정수 속도 1.0 초과분 다음 턴 이월.
- 공격도 행동 1회 소비.

## 전투 피드백

명중 시 핏자국 + 피격 플래시.

- 명중 시 해당 타일에 `%` 핏자국 — 빨강 `rgba(0.8, 0, 0, 1.0)`.
- 핏자국 수명 15~30 턴 랜덤. 플레이어 행동마다 `1/수명` 알파 감소,
  0 이면 소멸.
- 핏자국 Z = `Z_BLOOD = 0.5` (바닥 0 위, 몬스터 0.8 아래).
- 피격 시 엔티티 `HIT_FLASH_DURATION = 0.15s` 동안 빨강. 끝나면 원색.
- 이미 플래시 중 재피격 시 타이머 초기화, 원색 보존.
- 몬스터 처치 후에도 핏자국 잔류. 플레이어 사망해도 플래시 자연 만료.

## 원소 반응

전투에 원소 속성 도입. 두 원소 중첩 시 반응 발동.

### 원소

| 원소 | 지속 | 지속 효과 | 부여 |
|------|------|----------|------|
| 불 | 3턴 | 매 턴 2 피해 | 검 공격 40% |
| 얼음 | 4턴 | 이동 속도 50% 감소 | 창 공격 40% |
| 독 | 5턴 | 매 턴 1 피해 | 활 공격 (구) — 현재 활은 번개 |
| 번개 | 1턴 | 없음 (반응 트리거) | 활 공격 40% |

무기-원소: 검→불, 창→얼음, 활→번개. 몬스터 속성: 고블린→독(35%),
오크→불(35%), 트롤→얼음(35%).

### 반응 (두 번째 원소 부여 즉시)
- 불+얼음 = **융해**: 15 피해, 두 원소 제거.
- 불+독 = **독기**: 10 피해.
- 얼음+번개 = **파쇄**: 20 피해.
- 얼음+독 = **동상**: 5 피해 + 2턴 행동불능 (몬스터 only).
- 불+번개 = **플라즈마**: 8 피해 + 3턴 행동불능 (몬스터 only).
- 독+번개 = **전기독**: 8 피해 + 강화 독(6턴) 부여.

### 동작
- 같은 원소 재부여 시 지속 시간만 갱신.
- DoT 는 `PlayerActedEvent` 마다 1회.
- 불·독 DoT 로 HP ≤ 0 → 플레이어에게 `Defeated`.
- `Stunned` 는 몬스터에게만. Stunned 중 행동 건너뜀.
- 얼음 부여 시 이동 속도 절반, 만료 시 원래 복원.
- 부여·반응 발동 시 로그.
- 몬스터 사망 시 원소 효과 함께 소멸 (despawn).
- 얼음 중복 부여해도 속도 추가 감소 없음.

### 아키텍처
```
ElementalPlugin
  apply_elements          ElementalApplyEvent → 추가 or 반응
  process_elemental_turns PlayerActedEvent 마다 DoT/만료
  tick_stunned            행동불능 카운트다운

ElementalStatus     활성 원소 → 남은 턴 (Component)
Stunned             몬스터 행동불능 (Component)
Slowed              얼음 속도 감소, 원래 속도 보존 (Component)
ElementalApplyEvent 부여 요청 (Event)
```

## 투사체 (Projectile)

화살은 직사 궤적 (탑다운).

### 동작
- 활 장착 + 원격 모드에서 발사.
- 발사자 → 목표 타일 직선 궤적.
- `GravityScale(0.0)` — 중력 없음.
- 진행 방향으로 회전 (`rotate_arrow`).
- 몬스터 타일 진입 시 데미지 + 즉시 소멸.
- 벽 타일 진입 시 소멸 (관통 X).
- `ARROW_LIFETIME = 2.0s` 만료 시 자동 소멸.
- 발사는 턴 소비 (`PlayerActedEvent`).
- 활 사거리 = `BOW_RANGE = 8` 타일.
- 활 = 번개 원소 40% 부여.

### 상수
- `BOW_RANGE = 8`.
- `ARROW_LIFETIME = 2.0`.
- `ARROW_SIZE = Vec2::new(8.0, 3.0)`.
- `ARROW_COLOR = rgb(0.8, 0.6, 0.2)` (황갈).
- 화살 Z = `0.9` (몬스터 0.8 위).

### 속도 계산
```
flight_time = 0.3 + distance / 400.0
vx = Δx / flight_time
vy = Δy / flight_time
```
이전 곡사식 `vy = (Δy + 0.5 g t²) / t` 는 제거 — 직사 변경.

### 트리거
1. 활 장착.
2. 클릭 타일에 몬스터 (`MonsterTiles`).
3. 클릭 타일이 FOV 내 가시 (`map.tiles[idx].visible`).
4. 사거리 ≤ 8 타일.

### 엣지
- 인접 타일 몬스터에게도 발사 (근거리).
- 비행 중 몬스터 이동 시 빗나감.
- 활 미장착 → 기존 근접 공격.
- 모달 (장비/상점/도움말) 열린 상태 발사 불가.

## 원격 공격 (키보드 + 마우스 통합)

활 장착 시 모드 진입 → 키보드 또는 마우스로 타겟 → 발사. 모드 외부엔
즉시 발사 단축 없음 (모든 발사는 F 모드 경유).

### 키매핑
- `F` — 원격 모드 진입 (활 장착 시만; 아니면 안내 로그).
- `Tab` / `Shift+Tab` — 다음/이전 적. FOV+사거리 안 적, 거리 오름차순,
  거리 같으면 (tile_x, tile_y) tie-break.
- `←→↑↓` / `WASD` — 자유 커서 한 칸 이동 (대각 가능).
- 마우스 이동 (`CursorMoved`) — 커서를 마우스 위치 타일로 갱신. 키보드와
  공존: 키보드 조작 후 마우스가 움직이지 않으면 그 위치 유지.
- `Enter` / 좌클릭 — 발사. 1턴 소비.
- `Esc` / 우클릭 — 취소 (턴 소비 X).
- 패널 (인벤토리/상점/도움말) 열린 동안 키·마우스 모두 무시.

### 발사 판정
- 사거리 초과 → 무시 + "사거리를 벗어났다."
- LoS 차단 (`is_line_of_sight_clear`) → 무시 + "장애물에 막혔다."
- 타겟 타일에 적 있으면 적중. 없으면 그냥 화살이 날아가서 사라짐.
- 발사 성공 시 `PlayerActedEvent`.

### 시각 표시
- 모드 진입 시 커서 entity 1개 spawn (Text2d, glyph `+`).
- 매 프레임 cursor 타일 따라 갱신.
- 사거리 OK + LoS OK = 노랑, 불가 = 빨강.
- 모드 종료 시 entity despawn.

### 자동 타겟
- 진입 시 가장 가까운 적 자동 타겟. 없으면 player 위치.
- Tab 순환은 즉석 후보 목록 계산 (적 수 적어 부담 없음).
- 자유 커서 이동 후에도 Tab 누르면 적 목록으로 복귀.

### 데이터 구조
```rust
#[derive(Resource, Default)]
pub struct RangedTargeting { pub active: bool, pub cursor: (usize, usize) }

#[derive(Component)] pub struct RangedCursor;
```

### 모듈 구성
- `src/modules/ranged/mod.rs` + `RangedPlugin` (`main.rs` 등록).
- `player_movement` 가 `RangedTargeting.active` 시 일반 이동 입력 차단.
- `on_mouse_click` 도 `ranged.active` 시 path 이동 차단.

### MVP 범위
- 화살 자원 무한 (TODO).
- 데미지 = `weapon_attack(BOW, &items)`, Lightning 원소.
- 사거리별 명중률 보정 없음.

### 테스트
- `sorted_targets_*` — 거리순 정렬 / out-of-range 제외 / unseen 제외.
- 활 미장착 시 F 무시.
- 시각: 모드 진입/탈출, Tab 순환, 발사 성공/실패 로그.

## 물리 기반 파티클

`bevy_rapier2d` 를 파티클 이펙트 전용으로. 게임 로직은 턴제 그대로.

### 동작
- 명중 시 피격 위치에서 8개 파티클 방사형 분출.
- 무작위 방향 (0~360°) + 속도 30~120 px/s.
- 3×3px 붉은 사각형 `rgba(0.85, 0.05, 0.05, 1.0)`.
- 0.3초 동안 알파 선형 감소 후 소멸.
- `Sensor` — 게임 엔티티와 충돌 X (순수 시각).
- `BloodStain` 과 독립.
- `RapierPhysicsPlugin` 은 파티클 외 게임 엔티티에 영향 X.

### 탑다운 자연화
중력 낙하는 어색.
- `GravityScale(2.0)` → `GravityScale(0.0)`.
- `LinearDamping` 추가 — 자연스러운 정지.
- `PARTICLE_LIFETIME 0.45 → 0.3`.
- 향후: 공격 방향 (attacker → target) 받아 cone 분포 (시그니처 변경
  필요해 별도).

### 상수
- `PARTICLE_COUNT = 8`.
- `PARTICLE_LIFETIME = 0.3`.
- `PARTICLE_SIZE = 3.0`.
- `pixels_per_meter = 100.0` (Rapier).
- 파티클 Z = `Z_BLOOD + 0.1 = 0.6`.

### 엣지
- 파티클 수명은 실시간 (초) — 턴과 무관.
- 화면 밖이라도 수명 다하면 소멸.
