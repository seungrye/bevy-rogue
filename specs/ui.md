# UI

## 인게임 도움말 오버레이

외부 문서 없이 게임 안에서 핵심 조작을 확인할 수 있게 한다.

### 동작 명세

- `H` 또는 `?` (Shift+/) 로 도움말 오버레이를 토글한다.
- 도움말이 열린 상태에서 `Esc` 로 닫는다.
- 도움말 표시 상태는 `HelpPanelOpen` 리소스로 관리한다.
- 도움말이 열린 동안 키보드 이동·마우스 자동 경로 입력을 차단한다 — 다른
  모달과 같은 패턴.

### 표시 내용 (그룹)

- 이동·대기
- 전투·탐험 (원격 공격 키 포함)
- 패널 (장비, 퀘스트, 상점, 미니맵)
- Game Over 조작
- 개발/디버그 키 (F1 맵 생성기, G 글리프 스타일 등)

### 구현 위치
- `src/modules/ui/help.rs`
- `src/modules/ui/mod.rs`
- 입력 차단: `src/modules/player/mod.rs`

### 테스트
- `help_text_lists_core_commands`
- 전체 회귀: `cargo test`

### 남은 개선 후보
- 키 바인딩을 단일 데이터 소스로 관리해 문서·오버레이 중복 제거
- 카테고리 탭 또는 스크롤
- 컨텍스트 도움말 (현재 상황 키만 강조)
