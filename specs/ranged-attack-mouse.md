# 원격 공격 — 마우스 타겟 통합 + 즉시 발사 제거

## 배경
원격 공격 키보드 인터페이스 구현 후, 타겟을 마우스로도 선택할 수 있게
한다. 동시에 기존 `on_mouse_click` 의 "활 + 적 타일 즉시 발사" 단축은
일관성을 위해 제거한다 — 모든 발사는 F 모드를 거치도록 통일.

## 동작 명세

### 모드 외부
- 활을 장착해도 `F` 누르기 전에는 발사할 수 없다.
- 적 타일을 마우스로 클릭하면 그냥 자동 경로 이동 (현재 fallback 동작).

### 모드 내부 (`RangedTargeting.active == true`)
- **마우스 이동** (`CursorMoved` 이벤트) — 커서 위치를 마우스가 가리키는
  타일로 갱신. 키보드 자유 커서/Tab 입력과 공존: 키보드 조작 후 마우스가
  움직이지 않으면 그 위치 유지.
- **좌클릭** — 발사. 사거리·LoS 검사 후 `FireProjectileEvent` 발행.
  Enter 와 동일.
- **우클릭** — 취소. Esc 와 동일.
- 마우스 좌표가 게임 화면 밖이거나 맵 범위를 벗어나면 무시.

### 패널 차단
인벤토리/상점/도움말 열린 동안에는 마우스 처리도 무시 (기존 키보드 입력
차단 패턴과 동일).

## 구현 메모
- 새 시스템 `handle_ranged_mouse` 추가 — `RangedTargeting.active` 검사
  후 `CursorMoved` 이벤트 / `MouseButton` 입력 처리.
- `on_mouse_click` (`src/modules/player/mod.rs`):
  - 활 장착 + 적 타일 즉시 발사 분기 삭제.
  - 모드 활성 중에는 path 이동도 막아야 — `if ranged.active { return; }`.
- 카메라/윈도우/맵 에서 마우스 좌표 → 타일 변환은 기존 `on_mouse_click`
  코드 재사용.
- 발사 시 cursor entity despawn — 키보드 흐름과 동일.
