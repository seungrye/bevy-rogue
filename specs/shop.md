# 상점 시스템

## 목적

마을의 상인 NPC와 상호작용해 아이템을 구매하거나 인벤토리 아이템을 판매한다.
코드 재컴파일 없이 상점 목록을 `SHOP_CATALOG` 상수로 관리한다.

## 동작 명세

### 금화 (Gold)

- [x] `PlayerInventory.gold: u32` — 플레이어 보유 금화
- [x] 게임 시작 시 기본 50G 보유
- [x] `earn_gold(amount)` / `spend_gold(amount) -> bool` 메서드 제공

### 상점 열기·닫기

- [x] 마을 상인(`상인` NPC)과 부딪히면 상점 패널이 열린다
- [x] Esc로 닫는다
- [x] 상점이 열려 있는 동안 플레이어 이동 입력은 무시된다

### 상점 패널 UI

- [x] 화면 중앙 고정, 너비 280px
- [x] 배경: 다크 그린 `rgba(0, 0.05, 0, 0.97)` (다른 패널과 동일)
- [x] Tab으로 구매/판매 탭 전환
- [x] ↑↓로 항목 이동, Enter로 구매/판매 실행
- [x] 패널 하단에 현재 보유 금화 표시

### 구매 탭

- [x] `SHOP_CATALOG` 의 모든 아이템을 이름·가격과 함께 표시
- [x] 보유 금화 부족 시 구매 불가 (로그에 메시지)
- [x] 구매 성공 시 금화 차감 + 인벤토리에 추가 + 로그 메시지

### 판매 탭

- [x] 인벤토리 내 퀘스트 아이템 제외한 모든 아이템 표시
- [x] 소비 아이템은 개수(xN) 표시, 1개씩 판매
- [x] 판매 성공 시 인벤토리에서 제거 + 금화 획득 + 로그 메시지
- [x] 판매할 아이템이 없으면 "판매할 아이템이 없습니다" 표시

## 상점 목록 (`SHOP_CATALOG`)

| 아이템 | 구매가 | 판매가 |
|--------|--------|--------|
| 체력 물약 | 50G | 25G |
| 검 | 100G | 50G |
| 창 | 150G | 75G |
| 활 | 80G | 40G |
| 가죽 갑옷 | 100G | 50G |

## 구현 세부사항

- `ShopOpenEvent` — 상인 NPC 충돌 시 발행
- `ShopPanelOpen(bool)` 리소스 — 패널 표시 여부
- `ShopUiState { cursor, mode: ShopMode }` 리소스
- `ShopFont(Handle<Font>)` 리소스 — 한글 폰트 핸들 보존 (동적 TextSection 생성 시 사용)
- `src/modules/ui/shop.rs` — ShopPlugin, 패널 렌더·입력 처리
- `villager::handle_bump` 에서 `name == "상인"` 분기 → `ShopOpenEvent` 발행
- 맵 생성기 순환 단축키: `Tab` → `F1` (Tab은 상점 탭 전환에 사용)
