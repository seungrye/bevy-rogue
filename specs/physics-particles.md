# 물리 기반 파티클 이펙트

## 목적

`bevy_rapier2d` 물리 엔진을 파티클 이펙트 전용으로 적용하여, 전투 피격 시 피 튀김 효과를 시각적으로 강화한다. 핵심 게임 로직(이동, 전투 판정, AI)은 기존 턴제 구조를 그대로 유지한다.

## 동작 명세

- [ ] 공격 명중 시 피격 위치에서 파티클 8개가 방사형으로 분출된다
- [ ] 파티클은 무작위 방향(0~360°)과 속도(30~120 px/s)로 초기 속도를 갖는다
- [ ] 파티클은 중력(`GravityScale(2.0)`)의 영향을 받아 포물선 궤적으로 낙하한다
- [ ] 파티클은 3×3px 크기의 붉은색(`rgba(0.85, 0.05, 0.05, 1.0)`) 사각형 스프라이트로 표시된다
- [ ] 파티클은 0.45초 동안 알파값이 선형으로 감소하다 소멸한다
- [ ] 파티클은 `Sensor`로 설정되어 게임 엔티티와 충돌하지 않는다 (순수 시각 효과)
- [ ] 파티클은 기존 핏자국(`BloodStain`)과 독립적으로 동작한다
- [ ] `RapierPhysicsPlugin`은 파티클 외 게임 엔티티(플레이어, 몬스터, 타일)에 영향을 주지 않는다

## 상수

- `PARTICLE_COUNT = 8` (피격 1회당 파티클 수)
- `PARTICLE_LIFETIME = 0.45` (초)
- `PARTICLE_SIZE = 3.0` (픽셀)
- `pixels_per_meter = 100.0` (Rapier 단위 기준)
- 파티클 Z 레이어: `Z_BLOOD + 0.1 = 0.6`

## 엣지 케이스

- 파티클 수명은 실시간(초)으로 관리되며 턴 이벤트와 무관하게 소멸한다
- 파티클이 화면 밖으로 나가더라도 수명이 다하면 자동 소멸한다
