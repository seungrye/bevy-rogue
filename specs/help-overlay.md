# 인게임 도움말 오버레이

## 목적

외부 문서를 열지 않고도 게임 안에서 핵심 조작을 확인할 수 있게 한다.
조작 수가 늘어도 신규 플레이어가 이동, 전투, 패널, 미니맵, Game Over 조작을 빠르게 발견할 수 있어야 한다.

## 동작 명세

### 열기와 닫기

- [x] `H` 로 도움말 오버레이를 열고 닫는다
- [x] `Shift + /` 조합을 `?` 입력으로 보고 도움말 오버레이를 열고 닫는다
- [x] 도움말이 열린 상태에서 `Esc` 를 누르면 닫는다
- [x] 도움말 표시 상태는 `HelpPanelOpen` 리소스로 관리한다

### 표시 내용

- [x] 이동과 대기 조작 표시
- [x] 전투와 탐험 조작 표시
- [x] 장비, 퀘스트, 상점, 미니맵 패널 조작 표시
- [x] Game Over 조작 표시
- [x] 개발/디버그성 전환 키 표시

### 입력 차단

- [x] 도움말이 열린 동안 키보드 이동 입력을 차단한다
- [x] 도움말이 열린 동안 마우스 자동 이동 경로 생성을 차단한다
- [x] 장비/상점 패널과 같은 기존 모달 입력 차단 규칙과 같은 방식으로 동작한다
