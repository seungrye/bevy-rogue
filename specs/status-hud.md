# 상단 상태 HUD

## 목적

플레이어가 매 턴 판단해야 하는 핵심 상태를 항상 한 줄로 확인할 수 있게 한다.
상세 정보는 장비/퀘스트/상점 패널에 남기고, HUD는 빠르게 훑는 전투·탐험 요약에 집중한다.

## 동작 명세

### 표시 위치

- [x] 맵 상단에 고정 높이 HUD를 배치한다
- [x] 장비/퀘스트/상점 패널보다 낮은 z-index를 사용해 패널이 열렸을 때 뒤에 깔리게 한다
- [x] 맵 너비 기준으로 배치한다

### 표시 내용

- [x] 현재 존 이름 표시
- [x] 누적 턴 표시
- [x] 레벨과 현재 XP/다음 레벨 필요 XP 표시
- [x] HP/MP 숫자 표시
- [x] 공격력/방어력 표시
- [x] 골드 표시
- [x] 장착 무기/방어구 표시
- [x] 현재 맵 생성기 이름 표시

### 갱신 조건

- [x] 존이 바뀌면 갱신한다
- [x] 턴이 바뀌면 갱신한다
- [x] 맵, 인벤토리, 장비, 성장 상태, 플레이어 스탯이 바뀌면 갱신한다
- [x] HUD 문자열 생성은 순수 함수로 분리해 테스트한다
