# 전투 피드백

## 목적

공격 명중 시 핏자국과 피격 플래시로 시각적 피드백을 제공하여 전투 타격감을 높인다.

## 동작 명세

- [ ] 공격이 명중하면 해당 타일에 `%` 핏자국이 생성된다
- [ ] 핏자국은 빨간색(`rgba(0.8, 0.0, 0.0, 1.0)`)으로 표시된다
- [ ] 핏자국은 알파값이 `BLOOD_DECAY_RATE = 0.25`/초로 감소하다가 0이 되면 소멸한다
- [ ] 핏자국은 Z 레이어 `0.5`에 위치한다 (바닥(0) 위, 몬스터(0.8) 아래)
- [ ] 피격 시 해당 엔티티가 `HIT_FLASH_DURATION = 0.15`초 동안 빨간색(`rgb(1.0, 0.0, 0.0)`)으로 표시된다
- [ ] 피격 플래시가 끝나면 원래 색상으로 복원된다
- [ ] 이미 플래시 중인 엔티티가 다시 피격되면 플래시 타이머가 초기화되고 원래 색상은 보존된다

## 상수

- `BLOOD_DECAY_RATE = 0.25` (알파/초, 초기값 1.0 → 약 4초 후 소멸)
- `HIT_FLASH_DURATION = 0.15` (초)
- 핏자국 Z 레이어: `Z_BLOOD = 0.5`

## 엣지 케이스

- 몬스터가 처치되더라도 핏자국은 해당 위치에 잔류하다 소멸한다
- 플레이어가 사망해도 플래시 컴포넌트는 자연 만료된다
