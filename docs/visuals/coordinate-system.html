좌표계: 타일 그리드 ↔ Bevy 월드 픽셀

맵 80×50, TILE_SIZE = 16. 그리드 좌표 (x,y) 와 Bevy 월드 좌표 (wx,wy) 의 변환.
출처: src/modules/map/mod.rs — tile_to_world_coords / world_to_tile_coords / MAP_WIDTH=80 / MAP_HEIGHT=50

축소도 (실제 80×50의 일부 격자만 표시). 칸에 마우스를 올리면 변환값이 갱신됩니다.
Bevy 월드는 +y가 위쪽, 원점(0,0)은 맵 중앙. 타일 (x,y)는 좌하단이 (0,0) 방향(코드의 y는 위로 증가).

변환 공식

ow = (80 * 16)/2 - 16/2 = 632 oh = (50 * 16)/2 - 16/2 = 392 tile_to_world(x, y): wx = x*16 - ow wy = y*16 - oh world_to_tile(wx, wy): x = floor((wx + ow + 8) / 16) y = floor((wy + oh + 8) / 16) x = clamp(x, 0, 79) y = clamp(y, 0, 49)
