티어별 무기·방어구 스탯 범위

각 막대는 드롭 시 롤되는 [min, max] 범위. 무기=공격력, 방어구=방어보너스. tier 1~5.

레어도 백분위 임계

레어도는 롤된 스탯이 [min, max] 안에서 차지하는 백분위 p = (rolled−min)/(max−min) 로 결정. 높게 롤될수록 높은 등급. 임계는 Rarity::from_roll 그대로.

참고: max ≤ min 이면 분모 0 → 백분위 0 → 일반(Common) 처리.

레벨별 티어 드롭 가중치

플레이어 레벨 L 의 "티어 밴드 중심" center = clamp(1 + (L−1)/3, 1, 5) 기준으로, d = item_tier − center 에 따라 가중치가 정해진다(tier_weight). 슬라이더로 레벨을 바꿔 보세요.

가중치 표(tier_weight): d≥+2 → 0.0 (게이트), +1 → 0.5, 0 → 1.0, −1 → 0.6, −2 → 0.3, ≤−3 → 0.1. 가중치 0 인 티어는 추첨 후보에서 제외됩니다.
