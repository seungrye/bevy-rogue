방향 시야 (Directional FOV)

facing 방향으로 큰 반원(FOV_FRONT), 등 뒤로 작은 반원(FOV_BACK) + 시선(LoS, 벽 차단). 잠입 시스템의 핵심.
출처: src/modules/map/mod.rs — is_in_view / is_line_of_sight_clear / FOV_FRONT=8 / FOV_BACK=3

조작

방향키 또는 아래 버튼으로 facing 변경 (8방향).
그리드를 클릭하면 그 칸의 벽을 켜고/끄기.
플레이어를 드래그(또는 Shift+클릭)하면 위치 이동.

정면 가시 타일 (반경 ≤ 8)
후면 가시 타일 (반경 ≤ 3)
벽 (이동·시야 차단)
플레이어 + facing 화살표
보이지 않는 바닥
규칙: front = dot(t−p, facing) ≥ 0 (수직 dot==0 도 정면). 정면이면 반경 8, 후면이면 반경 3. dist² ≤ r² 그리고 Bresenham 시선이 벽에 막히지 않아야 가시. facing이 (0,0)이면 방향 개념 없이 전방향 원형(반경 8)으로 폴백.
