# 키 바인딩 레퍼런스

## 도움말

| 키 | 동작 |
|----|------|
| `?` 또는 `H` | 인게임 도움말 열기/닫기 (조작 안내) |
| `Esc` | 도움말 닫기 |

## 피처 도감 / 가이드

| 키 | 동작 |
|----|------|
| `F2` | 피처 도감(가이드) 패널 열기/닫기 |
| `↑` `↓` | 도감 항목 선택 (경계에서 순환) |
| `Esc` | 도감 닫기 |

- 게임의 실제 기능(방향 시야·잠입, 조명/그림자, 원소 반응, 아이템 티어·레어도,
  파밍·드롭, 액티브 스킬, 함정, 파괴 지형·폭발, 고정 상점, 전투, 퀘스트/저널)을
  카테고리·항목으로 정리한 **읽기 전용** 도감이다. 선택한 항목의 설명·간단 시각화가 펼쳐진다.
- 도움말(`H`/`?`)은 조작키 안내, 도감(`F2`)은 메커니즘 설명으로 역할이 나뉜다.

## 이동

| 키 | 동작 |
|----|------|
| `←` `→` `↑` `↓` 또는 `A` `D` `W` `S` | 한 칸 이동 (누르면 즉시, 120ms 후 연속 이동) |
| 대각 방향 (예: `→` + `↑` 동시) | 대각선 이동 |

- 연속 이동 중 방향 전환은 즉시 반영된다 (멈칫 없음)

## 대기

| 키 | 동작 |
|----|------|
| `Space` | 제자리 대기 — 이동 없이 턴 소비 (몬스터 행동·핏자국 소멸 등 진행) |

## 전투

| 상황 | 동작 |
|------|------|
| 몬스터 위치로 이동 | 자동으로 근접 공격 |

## 액티브 스킬 (MP 소모)

| 키 | 동작 |
|----|------|
| `1` | 파이어볼 — 조준 모드 진입 (MP 8) |
| `2` | 점멸(Blink) — 조준 모드 진입 (MP 5) |
| `3` | 치유 — 즉시 시전, HP 회복 (MP 6) |

- 조준 모드: `←` `→` `↑` `↓` 또는 `A` `D` `W` `S` 로 커서 이동, `Enter` 로 시전, `Esc` 로 취소
- 파이어볼: 조준 타일에 폭발(지형 파괴 + 범위 피해) + 반경 내 몬스터에 화염 원소
- 점멸: 시야·사거리 내 빈(통과 가능) 타일로 순간이동
- MP 부족 시 시전 불가(로그 표시), 시전 성공 시 MP 차감 + 턴 소비

## 함정 설치/해제

| 키 | 동작 |
|----|------|
| `T` | 함정 키트로 정면 타일에 플레이어 함정 설치 (키트 1개 소모) |
| `Y` | 인접·현재 타일의 노출된 함정 해제 |

- 설치: 정면 타일이 비어(통과 가능) 있고 점유되지 않아야 한다. 함정 키트(`trap_kit`)가 있어야 하며 설치 시 1개 소모.
- 플레이어 함정은 **몬스터 진입 시에만 발동**한다(플레이어 자신은 밟아도 발동 안 함).
- 해제: 해제 도구(`disarm_tool`) 보유 시 확정, 없으면 50% 확률. 성공 시 회수형 키트를 회수(키트 +1).
- 함정 키트·해제 도구는 소모품 인벤토리에 들어가며, 장비 패널의 "사용(회복)" 으로는 소비되지 않는다.

## 미니맵

| 키 | 동작 |
|----|------|
| `M` | 미니맵 표시/숨김 전환 |
| `Ctrl` + `-` 또는 `Ctrl` + Numpad `-` | 줌아웃 — 표시 반경 +5타일 (최대 70, 맵 전체 조망 가능) |

## 맵 생성

| 키 | 동작 |
|----|------|
| `F1` | 다음 맵 생성기로 순환하며 즉시 재생성 |
| `G` | 아이템 글리프 스타일 순환 (ASCII → 유니코드 → RPG 아이콘) |

- 커맨드라인: `--algorithm <이름>` (`-a`) 으로 시작 생성기 지정 (예: `--algorithm bsp`).
  레지스트리에 등록된 23종 모두 지정 가능
- `--help` (`-h`) 로 옵션과 생성기 안내 확인

## 퀘스트

| 키 | 동작 |
|----|------|
| `Q` 또는 `J` | 퀘스트 패널 / 저널 열기/닫기 |

- 패널 상단의 **저널** 요약은 진행 중(수락했고 아직 완료되지 않은) 퀘스트의 현재 목표만 읽기 전용으로 나열한다 (미시작·완료 퀘스트 제외).

## 장비 관리

| 키 | 동작 |
|----|------|
| `E` | 장비 관리 패널 열기/닫기 |
| `↑` `↓` | 인벤토리 항목 선택 |
| `Enter` | 선택한 무기·방어구·**액세서리** 장착/해제, 소모품 사용 |
| `Esc` 또는 `E` | 장비 패널 닫기 |

- 패널이 열려 있는 동안 이동이 차단된다 (턴 소비 없음)
- **액세서리 슬롯**: `scout_lens`(올빼미 안경) 착용 시 가드 시야가 빨갛게, `trap_scope`(광부의 등불) 착용 시 시야 반경 8 안의 숨김 함정 자동 노출.

## 상점

| 키 | 동작 |
|----|------|
| 가판대(`=`) 또는 상인(`$`)에 부딪힘 | 상점 패널 열기 |
| `Tab` | 구매 / 판매 탭 전환 |
| `↑` `↓` | 항목 선택 |
| `Enter` | 구매 또는 판매 실행 |
| `Esc` | 상점 닫기 |

- 상인(`$`)은 마을 가판대(카운터 `=`) 뒤에 **고정**되어 돌아다니지 않는다.
- 카운터는 통행 불가다. 카운터 **앞 타일에서** 카운터(또는 그 너머 상인)를 향해 이동하면 상점이 열린다.
- 상점이 열려 있는 동안 이동이 차단된다

## 대화창 / 트리거

| 키 | 동작 |
|----|------|
| `Esc` 또는 이동키 | 열린 대화창·트리거 패널 닫기 |

## 게임 오버

| 키 | 동작 |
|----|------|
| `R` 또는 `N` | 기존 세이브를 삭제하고 새 게임 시작 |
| `Esc` | 게임 종료 |
